## Technology Stack

### Core Infrastructure
- **Database:** ClickHouse (primary), PostgreSQL (metadata)
- **Orchestration:** Kubernetes with microk8s
- **Streaming:** Apache Kafka (optional add-on)
- **Monitoring:** Prometheus + Grafana
- **CI/CD:** GitHub Actions

### AI & Agents
- **LLM Integration:** Multi-model support (Claude, GPT-4, GLM-5)
- **RAG System:** LlamaIndex + Qdrant
- **Agent Framework:** OpenAI Agents + custom orchestration
- **MCP Protocol:** Standardized tool integrations

### Developer Experience
- **API:** REST + GraphQL
- **SDKs:** Python, TypeScript, Go
- **CLI:** OpenClaw-based control plane
- **Documentation:** /llms.txt for AI-friendly docs
